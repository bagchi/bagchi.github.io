DCSL is looking for a new graduate research assistant to work on a security project, starting the beginning of the Spring semester of 2015. The project is for 1 year, and is open to a PhD (1st or 2nd year) or a Masters (with thesis, 1st year) student in CS or ECE. Interviewing for the position will start with immediate effect and for maximum consideration, the student should apply by December 19.

Project Details

The project will create behavioral profiles of users based on data that is collected by systems that they interact with. The systems are distributed and consequently have different parts of the profile each machine can contribute. The profile will be used to determine if the user is legitimate or is unauthorized, using some legitimate users stolen credentials.

Qualifications

         Strong Linux system-building skills

         Ability to process and analyze large volumes of data

         Knowledge of C and a scripting language.

         Machine learning algorithms (clustering, pattern classification)
